Carousel Twitter Bootstrap v4
=============================
Carousel as implemented in Twitter Bootstrap V4 example.

Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist linil/yii2-tb4carousel "*"
```

or add

```
"linil/yii2-tb4carousel": "*"
```

to the require section of your `composer.json` file.


Usage
-----

Once the extension is installed, simply use it in your code by  :

```php
<?= \linil\tb4carousel\AutoloadExample::widget(); ?>```
